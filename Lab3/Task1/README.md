# Task1

Программа выполняет умножение матрицы на вектор (`double`) с параллельной инициализацией и вычисляет ускорение.

## Сборка

```bash
cmake -S . -B build
cmake --build build -j
```

## Запуск по условиям задания

```bash
cd build
./Task1/task1_benchmark
```

По умолчанию используются размеры `20000, 40000` и потоки `1,2,4,7,8,16,20,40`. Результаты выводятся в консоль.

## Быстрый локальный прогон

```bash
cd build
./Task1/task1_benchmark --quick
```
