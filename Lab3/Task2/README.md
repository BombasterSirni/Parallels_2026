# Task2

Реализовано клиент-серверное приложение:
- сервер: отдельный поток, очередь задач, контейнер результатов (`std::unordered_map`), шаблонный класс `TaskServer<T>`;
- клиенты: 3 потока (`sin`, `sqrt`, `pow`), каждый добавляет `N` задач, ждет результат и пишет в файл;
- тест: читает файлы и проверяет корректность вычислений.

## Сборка

```bash
cmake -S . -B build
cmake --build build -j
```

## Запуск приложения

```bash
cd build
./Task2/task2_app 100
```

Ограничение: `5 < N < 10000`.

Файлы результатов:
- `results/client_sin.txt`
- `results/client_sqrt.txt`
- `results/client_pow.txt`

## Запуск теста

```bash
cd build
./Task2/task2_test
```

Тест проверяет все строки файлов результатов и сравнивает с эталонными значениями (`sin/sqrt/pow`).
